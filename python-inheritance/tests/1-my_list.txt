The `MyList` class inherits from the built-in `list` class.

To use the class, first import it:
    >>> MyList = __import__('1-my_list').MyList

Now, you can create a MyList object:
    >>> my_list = MyList()
    >>> my_list.append(1)
    >>> my_list.append(4)
    >>> my_list.append(2)
    >>> my_list.append(3)
    >>> my_list.append(5)
    >>> print(my_list)
    [1, 4, 2, 3, 5]

The `print_sorted` method prints the list in ascending order:
    >>> my_list.print_sorted()
    [1, 2, 3, 4, 5]

The original list remains unchanged:
    >>> print(my_list)
    [1, 4, 2, 3, 5]
